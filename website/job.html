Ideal Job

UX Designer

User experience design is all about the process of seamlessly integrating a product for the use of a user. Along with User Interface design, I would help in shaping a product so that it is more usable and accessible to a wider audience. I am interested in this role as I have always been compelled by the strategies that different companies employ in order to create and attract people to their products. One such example is Apple with how they tailor their User experience in a way that that makes it difficult for users to transition away from their products because of how much of an insular environment they designed.

From a technical standpoint, interchangeable skills for UX design include graphic design and web development. Knowledge in HTML and CSS are also valued as well as the use of wireframing software. UX designers will also need to possess analytical skills and understand how to solve customer problems. They will have to think in the position of a customer in order to design user-centric experiences.

I have previously done a project in which my role was mainly focussed on the user experience. In that project I did numerous things that a UX designer is entailed to do. This included wireframing a prototype, constructing user personas, stories and communicating suggestions to my group members responsible for the development of the product. My E-Business Systems class last semester also taught me some rudimentary HTML and CSS skills which I can further develop.

Building a portfolio to showcase design skills and processes to employers. A portfolio will document all stages of a case study from discovery to solution. I will also learn to use multiple different wireframing software so that my skills are more flexible for employers looking for a specific skillset.
